Snake

Instructions

  • Press ENTER to start the game.
  • Press SPACEBAR to pause/continue the game.
  • Refresh the webpage to reset the game.

Controls

  • Player 1 uses ARROW keys to change snake direction.
  • Player 2 uses W, S, A, D keys to change snake direction.
  • Player 3 uses I, J, K, L keys to change snake direction.
  • Player 4 uses NUMPAD keys to change snake direction.
